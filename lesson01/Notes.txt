Q.1 What is HTML ?

Html is stands for Hyper text markup language, or a tool we use for create a website.

Html = Giving instructions to a computer, like: <h1>Hello World !</h1>

Explanation: I give a instruction to computer to create 'h1' inside 'Hello World !' text.
Now, lets open a file with google chrome and see the output. There will be a 'Heading'. See How pretty and straight-forward HTML is.
